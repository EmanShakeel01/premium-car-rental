# 🚗 Premium Car Rental Website

A modern, responsive, and user-friendly **Premium Car Rental Website** built with **React.js** and **Vite**. The application is designed to provide a seamless car rental experience, allowing users to browse premium vehicles, explore detailed information, and make bookings through an intuitive interface.

## Features

* Modern and responsive UI
* Premium vehicle categories
* Car listing with search functionality
* Filter cars by category
* Sort cars by price
* Detailed car information
* Booking form with validation
* Dynamic car data using JSON
* React Router navigation
* Loading and error states
* Reusable React components
* Mobile-first design
